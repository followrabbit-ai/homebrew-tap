# followrabbit CLI

Command-line tool for reviewing and optimizing cloud infrastructure costs with [Rabbit](https://followrabbit.ai).

## Install

### Homebrew (macOS/Linux)

```bash
brew install followrabbit-ai/tap/followrabbit
```

### Shell (macOS/Linux)

```bash
curl -fsSL https://followrabbit-ai.github.io/homebrew-tap/install.sh | sh
```

### npm (all platforms)

```bash
npm install -g @followrabbit/cli
```

## Verify

```bash
followrabbit version
```
